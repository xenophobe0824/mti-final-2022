声音问题

Condition

没有声音或音量过低。

Cause

1. 未进行音量设定。

2. 投影机和视频信号源之间的电缆未连接。

3. 视频设备设置出现问题。

Remedy

  1. 使用遥控器上的或按钮调节投影机的音量设定。
  2. 检查投影机和视频信号源之间的电缆连接。
  3. 检查以下投影机设置：①如果只是暂时停止，请按遥控器上的按钮恢复视频和音频。②按遥控器上的按钮以取消音频静音。
  4. 检查以下视频设备设置：①检查视频信号源，确保音量已调高并针对正确的信号源设置音频输出。②将已连接设备设为PCM输出。③使用HDMI电缆将投影机连接到MAC时，确保MAC支持通过HDMI端口连接的音频。
